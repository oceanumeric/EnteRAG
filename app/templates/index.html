AI-based Enterprise RAG

Welcome to the AI-based Enterprise RAG! While creating a Retrieval Augmented Generation (RAG) chat system may seem straightforward, scaling it to handle over 1.5 million documents (around 2000 books of 300 pages) presents a significant challenge. This project demonstrates how to build a scalable RAG system. It uses a huge dataset of book reviews from the Goodreads.

Key Features:

  • Review Upload: Users can upload review (could be any document) using the upload feature.
  • Query and Chat: Users can search and engage in conversations with books through the chat interface. It supports semantic search such as funny scifi for a rainy afternoon.
