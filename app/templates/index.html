AI-based Enterprise RAG

Welcome to the AI-based Enterprise RAG! While creating a Retrieval Augmented Generation (RAG) chat system may seem straightforward, scaling it to handle over 15 million documents presents a significant challenge. This project demonstrates how to build a scalable RAG system. It uses a huge dataset of book reviews from the Goodreads.

Key Features:

  • Document Upload: Users can upload documents using the upload feature.
  • Query and Chat: Users can search and engage in conversations with documents through the chat interface.
